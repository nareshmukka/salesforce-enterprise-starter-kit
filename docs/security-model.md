# Security Model (Baseline)

## Principles
- Least privilege by default
- Org-Wide Defaults (OWD) assumed private
- Permission Sets over Profiles
- Enforce CRUD/FLS in Apex and UI

## Org-Wide Defaults
- Account: Private
- Contact: Controlled by Parent
- Custom Objects: Private

## Access
- Permission Sets
- Permission Set Groups
- Apex Managed Sharing (when needed)
