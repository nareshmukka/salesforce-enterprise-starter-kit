# Naming Conventions

## Apex
- Classes: PascalCase (e.g., SharingService)
- Methods: camelCase (e.g., grantAccess)
- Tests: ClassNameTest

## LWC
- Component folder: camelCase (e.g., recordSummaryCard)

## Flows
- Record-triggered: RTF_<Object>_<Purpose>
- Screen flow: SF_<Purpose>
- Subflow: SUB_<Purpose>
