# Architecture Overview

This repository is an opinionated Salesforce “starter kit” showcasing enterprise patterns for:
- Data modeling
- Security & sharing
- Automation standards (Flow vs Apex)
- Integration patterns
- Deployment and release hygiene

## Design principles
- Security-first (least privilege)
- Bulk-safe Apex, testable code, clear naming
- Prefer configuration (Flow) where appropriate
- Centralized error handling and logging patterns
- Documentation as part of delivery
