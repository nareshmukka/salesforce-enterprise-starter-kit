# Salesforce Enterprise Starter Kit

Opinionated patterns and reference implementations for building enterprise-grade Salesforce solutions.

## What this repo demonstrates
- Security-first design and least-privilege access
- Clear naming and maintainable structure
- Flow vs Apex decision standards
- Reusable patterns for reliability and deployments

## Structure
- `docs/` – architecture and standards
- `force-app/` – Salesforce metadata (SFDX format)
